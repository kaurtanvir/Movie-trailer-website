# Movie trailer website
Movie trailer website project allows the users to review the movies,their storyline and watch the trailers on a web page.


## Features
•	Written in Python 2.7.13
•	No installation necessary just run the module and a html page will be created showing the movies.
•	Can be run on Windows/Mac.
•	Easy to use


## Installation
1.	Download the files(entertainment_center.py, media.py and fresh_tomatoes.py) from the zip folder.
2.	Make sure python is installed on the machine
3.	Run the module entertainment_center.py.
4.	You will see the html page with the movies listed and watch the trailer by clicking on the movie.
